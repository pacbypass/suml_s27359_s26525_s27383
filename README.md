# Klasyfikator rasy Psa/Kota na podstawie zdjęcia

## Wymagania

Komputer z systemem windows 10+ / ubuntu 20.04+ z dostępem do internetu

### Wymagane oprogramowanie
1. Python 3 (rekomendowana wersja 3.14.0)
2. pip3
3. git

## Instrukcja użytkownika

### Linux

korzystając z wbudowanego terminala w systemie, klikając przycisk "windows" wyszukujemy i znajdujemy "Terminal". W terminalu należy wpisać komendy znajdujące się poniżej.

```bash
git clone https://github.com/pacbypass/suml_s27359_s26525_s27383
cd suml_s27359_s26525_s27383
./run.sh
```

### Windows

Korzystając z cmd (włączane przez skrót win + r, a następnie wpisując cmd wykonujemy następujący zestaw komend


```bash
git clone https://github.com/pacbypass/suml_s27359_s26525_s27383
cd suml_s27359_s26525_s27383
run.cmd
```
